### Reference Model

Represents a reference used in chat responses, containing source information and display data.

---

#### `reference_num` *(integer)* <span style={{ color: 'red', fontWeight: 'bold' }}>Required</span>  
Reference number. Range: -2147483648 to 2147483647.

---

#### `source` *(string)* <span style={{ color: 'red', fontWeight: 'bold' }}>Required</span>  
Source of the reference. Minimum 1 character.

---

#### `showed_num` *(integer, Optional)*  
Number indicating how many times this reference was shown. Range: 0 to 2147483647.

---

#### Example

```json
{
  "reference_num": 0,
  "source": "string",
  "showed_num": 0
}
```
